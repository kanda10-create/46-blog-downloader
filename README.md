# 坂道 BLOG 下载工具

用于下载坂道系列（乃木坂46 / 樱坂46 / 日向坂46 / 榉坂46）  
官方 BLOG 的个人工具。

本工具适合用于 **资料整理 / 个人备份 / 翻译辅助**。

---

## 功能简介

- 按成员下载官方 BLOG
- 保存 BLOG 正文与官方配图
- 本地保存，操作简单
- 每个坂道工具彼此独立，可单独使用

---

## 可用工具一览

本仓库提供以下 4 个独立工具，请按需下载：

- **Nogizaka46 BLOG Downloader**  
  → 适用于：乃木坂46

- **Sakurazaka46 BLOG Downloader**  
  → 适用于：樱坂46

- **Hinatazaka46 BLOG Downloader**  
  → 适用于：日向坂46

- **Keyakizaka46 BLOG Downloader**  
  → 适用于：榉坂46

👉 所有工具均在 **Releases 页面** 提供下载。

---

## 如何选择下载哪个工具？

- 只需要乃木坂46 → 下载 **Nogizaka46** 版本  
- 只需要樱坂46 → 下载 **Sakurazaka46** 版本  
- 只需要日向坂46 → 下载 **Hinatazaka46** 版本  
- 需要榉坂46 → 下载 **Keyakizaka46** 版本  

每个工具互不依赖，无需全部下载。

---

## 使用方法

1. 在 Releases 页面下载对应工具的压缩包
2. 解压后运行 exe 文件
3. 选择成员
4. 设置保存路径
5. 点击开始下载

下载内容包括：
- BLOG 正文
- 官方配图
